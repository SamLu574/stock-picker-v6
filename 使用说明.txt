# A股形态选股系统 v6.0 - 使用手册

## 一、首次启动

1. **双击 `A股形态选股系统v6.0.exe`**，首次启动约需 15-30 秒（解压资源），请耐心等待。
2. 程序主界面分为三个标签页：**实时监控**、**形态选股**、**回溯验证**。
3. 建议先点击顶部 **"🔄 更新数据"** 按钮，获取最新行情数据（首次更新约 3-5 分钟）。

---

## 二、界面详解

### 📌 顶部工具栏

| 按钮 | 功能 |
|------|------|
| **🔄 更新数据** | 从东方财富下载最新交易日K线数据。如果数据库已有今天的数据，会提示"无需更新"。 |
| **▶ 执行选股** | 在"形态选股"标签页中，根据当前设置的参数立即筛选符合条件的股票。 |

---

### 📊 标签页 1：实时监控

用于盘中实时监控股票突破信号，触发时会有**声音提醒 + 弹窗**。

#### 左侧：监控设置

| 控件 | 说明 |
|------|------|
| **添加股票** | 输入框：输入股票代码（如 `600545` 或 `sh600545`），点右侧"+"号添加到监控池。 |
| **监控池列表** | 显示当前正在监控的股票代码。右键可删除。 |
| **突破天数 (N)** | 滑块：设置突破前 N 日高点的阈值。默认 60 天。数值越小越敏感。 |
| **止损比例 (%)** | 输入框：持仓下跌超过此比例时提醒卖出。默认 -5%。 |
| **止盈比例 (%)** | 输入框：持仓上涨超过此比例时提醒卖出。默认 10%。 |
| **▶ 启动 / ⏹ 停止** | 开始/停止实时监控。每 3 分钟自动刷新一次价格。 |

#### 右侧：持仓表

显示所有监控股票的实时状态：

| 列名 | 说明 |
|------|------|
| **代码** | 股票代码（纯数字） |
| **买入价** | 已买入股票的成交价格（黄色）。未买入显示 `-`。 |
| **现价** | 当前最新价格。 |
| **盈亏** | 已买入股票显示涨跌幅（红涨绿跌）。未买入显示距 N 日高点的差距百分比。 |
| **状态** | 显示"正常"、"突破"、"止损"、"止盈"等状态。 |

#### 底部：日志区

记录每次刷新的详细信息和触发的报警消息。

---

### 📈 标签页 2：形态选股

通过三层过滤筛选符合技术形态的股票。

#### 左侧：过滤条件

**L3 风险过滤**（排除高风险股票）
- ☑ **排除 ST**：勾选后剔除 ST/*ST 股票。
- ☑ **排除科创板**：勾选后剔除 688 开头的股票。
- ☑ **排除北交所**：勾选后剔除 8/4/9 开头的股票。

**L2 突破筛选**（技术面筛选）
- **半年涨幅上限 (%)**：过去 120 个交易日涨幅不能超过此值。默认 110%。避免追高。
- **突破天数 (N)**：突破前 N 日高点。默认 60。
- **选股方向**：下拉框选择"多头"（上涨趋势）或"空头"（下跌趋势）。

**L1 K线相似度**（形态匹配）
- **参考图片**：点击"选择图片"上传一张K线截图作为模板。
- **相似度阈值 (%)**：与参考图片的相似度必须达到此值。默认 73%。越高越严格。

#### 右侧：结果表

显示筛选通过的股票：

| 列名 | 说明 |
|------|------|
| **代码** | 股票代码 |
| **相似度** | 与参考图片的匹配度（蓝色） |
| **现价** | 当前价格 |
| **今日涨跌** | 今日涨跌幅（红涨绿跌） |

#### 底部：漏斗统计

显示每一层过滤的通过/淘汰数量，帮助调整参数。

---

### 📉 标签页 3：回溯验证

对历史数据进行回测，验证策略有效性。

#### 顶部：回测设置

| 控件 | 说明 |
|------|------|
| **起始日期** | 回测开始日期。格式：YYYY-MM-DD。 |
| **结束日期** | 回测结束日期。 |
| **初始资金** | 回测起始资金。默认 100000。 |
| **单次仓位** | 每次买入使用的资金比例。默认 10%（即每次用 10% 的资金买入一只股票）。 |
| **半年涨幅上限** | 同形态选股页。 |
| **突破天数 (N)** | 同形态选股页。 |
| **相似度阈值** | 同形态选股页。 |
| **参考图片** | 同形态选股页。 |

#### 中部：操作按钮

- **▶ 开始回测**：按设置运行回测。进度条显示当前进度。
- **📊 显示结果**：回测完成后点击查看详细报告。

#### 底部：结果展示

回测完成后显示：

**统计指标**
- **总收益率**：回测期间总盈利百分比。
- **最终资金**：回测结束时的账户总金额。
- **交易次数**：总共完成的买卖交易笔数。
- **胜率**：盈利交易占总交易的比例。

**收益曲线图**
- 显示每日账户资金变化曲线。

**交易明细表**
- 列出每笔交易的买入/卖出日期、价格、盈亏等信息。

## 三、常见问题

### Q1: 为什么启动这么慢？
A: exe 文件包含完整 Python 环境（约 270MB），首次启动需要解压到临时目录。后续启动会快一些。

### Q2: 点击"更新数据"卡在 0/0 怎么办？
A: 说明数据库已有今天的数据，无需更新。如果想强制重新下载，可以删除 `stock_data.db` 文件后重启程序。

### Q3: 监控时没有声音提醒？
A: 检查电脑音量是否开启。Windows 系统提示音可能被静音。

### Q4: 如何配置飞书通知？
A: 目前版本飞书 webhook 配置项暂未开放 UI 入口。如需启用，请联系开发者手动配置。

### Q5: 回测结果和实盘不一样？
A: 回测使用历史数据，假设"次日开盘价"成交。实盘受滑点、停牌、涨跌停限制影响，会有偏差。
